# react-native-screen-loading

Screen loading measurement in React Native

## Installation

```sh
npm install react-native-screen-loading
```

## Usage

```js
import { ScreenLoadingView } from "react-native-screen-loading";

// ...

<ScreenLoadingView color="tomato" />
```
